# 汽車維修費用計算系統

## 系統概述
這是一個專為汽車修配廠設計的Windows Forms應用程式，用於管理和計算汽車維修服務的費用，包括自動計算費用、生成報表和資料管理功能。

## 主要功能

### 1. 服務項目管理
系統提供三個主要服務類別，每項服務都有固定的收費標準：

#### 機油與潤滑服務
- 更換機油：NT$780
- 潤滑保養：NT$540

#### 清洗服務
- 水箱清洗：NT$900
- 變速箱清洗：NT$2,400

#### 其他維修服務
- 檢驗：NT$450
- 更換消音器：NT$3,000
- 輪胎換位：NT$600

### 2. 零件與工時計算
系統支援彈性的費用輸入機制：
- **零件費用輸入**：允許使用者輸入自訂的零件費用（以新台幣計算）
- **工時輸入**：提供工時數輸入功能
- **自動計算**：
  - 工資費用 = 工時 × NT$600/小時
  - 零件稅金 = 零件費用 × 6%

### 3. 費用計算與顯示
系統提供完整的費用計算功能：

#### 計算流程
1. **服務與工資總額** = 所有選定服務費用 + 工時費用
2. **稅金** = 零件費用 × 6%
3. **最終總費用** = 服務與工資總額 + 零件費用 + 稅金

#### 費用顯示
- 服務與工資：顯示服務費用加上工資費用的總額
- 零件費用：顯示輸入的零件費用
- 稅金：顯示零件的應納稅金
- 總費用：最終的結算金額

所有金額都以 `NT$###,###` 的格式顯示，使用千位分隔符號。

### 4. 資料管理功能
系統提供基本的資料操作功能：

#### 計算功能
- 點擊「計算費用」按鈕：根據選定的服務項目、輸入的零件費用和工時，自動計算所有費用
- 自動驗證輸入資料的合法性
- 即時更新費用顯示

#### 清除功能
- 點擊「清除」按鈕：重設所有輸入和選項
- 清除所有複選框的勾選狀態
- 清空零件費用和工時輸入
- 清空費用概要中的所有顯示

#### 資料驗證
系統實施完整的資料驗證機制：
- 驗證零件費用為有效的數字且非負數
- 驗證工時為有效的數字且非負數
- 無效輸入時提示使用者並將焦點移至錯誤欄位

### 5. 報表輸出功能
系統提供專業的報表匯出功能：

#### 報表內容
報表包含以下詳細資訊：
- 報表生成日期和時間
- 選定的所有服務項目清單
- 零件費用和工時資訊
- 詳細的費用計算明細
- 最終的總費用

#### 報表格式
```
========================================
        汽車維修費用計算報表
========================================
報表日期: 2024-03-05 14:30:45

【已選服務項目】
  ? 更換機油 (NT$780)
  ? 水箱清洗 (NT$900)
  ...

【零件與工時】
  零件費用: NT$5,000
  工時: 2 小時 @ NT$600/小時

【費用明細】
  服務費用: NT$1,680
  工資費用: NT$1,200
  ─────────────────
  服務與工資合計: NT$2,880

  零件費用: NT$5,000
  零件稅金 (6%): NT$300
  ═════════════════
  【最終總費用】: NT$8,180
========================================
```

#### 使用方式
1. 點擊「匯出報表」按鈕
2. 在檔案保存對話框中選擇保存位置
3. 自訂檔名（預設為 `維修報表_yyyyMMdd_HHmmss.txt`）
4. 點擊「保存」完成匯出

### 6. 應用結束時的提示
當使用者點擊「關閉」按鈕時：
- 系統詢問是否需要在結束前保存維修明細
- 使用者可選擇「是」以匯出報表、「否」直接關閉、或「取消」返回應用

## 程式架構

### 類別結構

#### RepairServiceConstants（維修服務常數類）
定義所有固定的服務費用和稅率常數，便於集中管理和日後修改。

#### CostCalculator（費用計算器）
提供費用計算的核心方法：
- `CalculateServiceCost()`：計算服務費用總額
- `CalculateLaborCost()`：計算工資費用
- `CalculatePartsTax()`：計算零件稅金
- `CalculateServiceAndLaborTotal()`：計算服務與工資合計
- `CalculateTotalCost()`：計算最終總費用
- `TryParseAmount()`：驗證和轉換輸入資料

#### RepairReport（維修報表）
表示一份完整的維修記錄，包含：
- 報表日期和時間
- 選定的服務項目清單
- 各項費用資訊
- `GenerateReportText()`：生成格式化的報表文本

#### Form1（主表單）
- `BtnCalculate_Click()`：計算費用
- `BtnClear_Click()`：清除資料
- `BtnExport_Click()`：匯出報表
- `BtnExit_Click()`：應用結束
- `CalculateSelectedServicesCost()`：計算選定服務的費用
- `ClearAllInputs()`：清除所有輸入
- `BuildRepairReport()`：構建報表物件

## 輸入驗證

系統實施以下驗證規則：

1. **零件費用**
   - 必須為有效的數字
   - 不能為負數
   - 空值將被視為 0

2. **工時**
   - 必須為有效的數字
   - 不能為負數
   - 空值將被視為 0

3. **計算時的驗證**
   - 無效的輸入將顯示警告訊息
   - 焦點將被移至錯誤欄位

## 技術規格

- **開發框架**：.NET 8
- **UI框架**：Windows Forms
- **編程語言**：C#
- **目標平台**：Windows

## 使用建議

1. **初次使用**
   - 選擇需要的服務項目（可複選）
   - 輸入零件費用和工時（如有）
   - 點擊「計算費用」檢視結果

2. **生成報表**
   - 完成費用計算後點擊「匯出報表」
   - 選擇合適的儲存位置
   - 報表將以文本格式保存

3. **重新計算**
   - 修改選項或輸入後，再次點擊「計算費用」即可更新結果
   - 點擊「清除」可快速重置所有資料

4. **應用結束**
   - 點擊「關閉」按鈕
   - 根據提示決定是否保存報表
